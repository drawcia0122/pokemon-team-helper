# ニュース・グッズ・イベント掲載方針

このページは非公式です。公式各社、掲載元、販売元とは関係ありません。

- 公式サイト、公式ショップなど出典が明確な情報を優先します。
- 本文や商品説明を転載せず、独自の短い紹介と元ページへのリンクだけを掲載します。
- 外部サイトの画像はコピー、保存、表示しません。
- 日付、価格、開催条件、在庫などは変更される場合があります。正確性や在庫を保証しません。
- 購入・応募・参加前に、必ず元ページで最新条件を確認してください。

## 自動収集データ

- 手動確認済みデータと自動生成データを分離し、手動データを常に優先します。
- 自動収集は、robots.txt、利用条件、公開された取得方法を確認できた公式ソースに限定します。
- 自動収集でも記事本文・画像は保存せず、メタデータ、短い独自案内、元ページへのリンクだけを掲載します。
- 規約が不明、またはサイト外利用が禁止されているソースには通信しません。
- 詳細は [公式ポケモン情報の収集仕様](./POKEMON_CONTENT_COLLECTION.md) を参照してください。
